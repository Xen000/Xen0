Le Cursus Pour Devenir Développeur Web

Pour devenir développeur web, il existe de nombreuses façons avec plus ou moins d'années d'études. Avant le bac, il n'existe pas forcément de parcours "spécifique". Vous pouvez très bien partir d'un bac pro ou d'un bac générale pour ensuite faire vos études.

De plus en plus de personnes choisissent de changer de métier pour devenir développeur web : pas de panique ! Il existe de nombreuses formations en ligne que vous pouvez faire depuis chez vous !

Post-Bac

Il existe de nombreuses possibilité pour devenir développeur web après le bac, je vais donc essayer d'être le plus précis possible.

  1. Niveau Bac + 2
    • BTS Système Numérique (SN)
    • BTS Services Informatiques aux Organisations (SIO)
    • Titre professionnel architecte intégrateur d'applications web
  2. Niveau Bac + 3
    • Licence informatique
    • Licence professionnelle métiers de l'informatique : applications web
    • Licence professionnelle métiers du design
    • BUT informatique
    • Titre professionnel concepteur développeur d'application web et mobile
  3. Niveau Bac + 5
